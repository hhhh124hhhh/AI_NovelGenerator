# 🚀 AI小说生成器2.0启动指南

## ✅ 正确启动方式

### 1. 使用 uv (推荐)
```bash
uv run python main.py
```

### 2. 标准Python启动
```bash
python main.py
```

### 3. Windows批处理文件
```bash
run.bat
```
